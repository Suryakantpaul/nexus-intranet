# Nexus — Corporate Intranet

A modern, feature-rich corporate intranet built with React + Vite for the June 2026 frontend assessment.

## Features
- **Dashboard** — Live pulse ticker, org stats, leadership message, activity feed, kudos leaderboard, upcoming events
- **Announcements** — Filterable feed with action-required, moderation-ready post creation
- **Recognition Hub** — Peer appreciation, new joiner carousel, gamified monthly leaderboard, Give Kudos modal
- **Teams & Verticals** — Department cards with progress indicators, org-wide milestone tracker
- **Knowledge Base** — HR/IT policies, discussion forum, topic-based browsing
- **Events & Calendar** — Interactive calendar with event indicators, filterable event list
- **Celebrations** — Birthday/anniversary/promotion posts with reactions and replies
- **Gallery** — Album-based photo grid with upload flow
- **People Directory** — Searchable, filterable employee directory

## Stack
- React 18, Vite 5
- Tabler Icons (webfont)
- Inter + Fraunces (Google Fonts)
- Pure CSS (no UI lib dependency)
- Zero runtime dependencies beyond React

## Deploy to Vercel
```bash
npm install
npm run build
# Push to GitHub, then import to Vercel
```

## Run locally
```bash
npm install
npm run dev
```
